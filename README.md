## 🧪 Run Locally

1. Clone the repository:
   ```bash
   git clone https://github.com/SohelHossain1218/sohelhossain.git
   ```
2. Open `index.html` in any modern browser.
3. No build steps or dependencies required.

> 💡 Make sure all images (`.jpg`) are placed in the same folder as `index.html` to avoid broken links.
